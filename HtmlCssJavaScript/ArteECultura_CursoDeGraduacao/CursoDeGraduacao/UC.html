Programação de soluções computacionais

A disciplina de Programação de Soluções Computacionais dentro da Engenharia de Software foca no desenvolvimento de software para resolver problemas específicos. Ela ensina os fundamentos da programação, incluindo o uso de diferentes linguagens e a criação de código eficiente. Os alunos aprendem a trabalhar com estruturas de dados, como listas e pilhas, e a implementar algoritmos para realizar tarefas como busca e ordenação.

A disciplina também abrange a programação orientada a objetos, permitindo a criação de software modular e reutilizável, e o uso de conceitos como encapsulamento e herança. Além disso, os estudantes são ensinados a planejar e representar algoritmos usando pseudocódigo e fluxogramas, e a usar ferramentas de desenvolvimento para escrever e testar o código. Em resumo, a disciplina prepara os alunos para desenvolver soluções de software eficazes e bem estruturadas.

Usabilidade, Desenvolvimento Web, Mobile e Jogos

A unidade curricular de Usabilidade, Desenvolvimento Web, Mobile e Jogos dentro da Engenharia de Software foca em criar aplicações eficazes e agradáveis em diferentes plataformas. Ela ensina a melhorar a usabilidade, garantindo que as interfaces sejam intuitivas e atendam às necessidades dos usuários. No desenvolvimento web, os alunos aprendem a construir sites e aplicações que funcionam bem em diversos navegadores e dispositivos usando tecnologias como HTML, CSS e JavaScript.

No desenvolvimento mobile, a disciplina cobre a criação de aplicativos para smartphones e tablets, abordando tanto plataformas específicas (Android e iOS) quanto soluções cross-platform. Por fim, o desenvolvimento de jogos explora a criação de jogos digitais, incluindo o design de níveis e a programação de mecânicas usando motores de jogos como Unity e Unreal Engine. Em resumo, a unidade prepara os alunos para desenvolver soluções de software interativas e eficazes para uma ampla gama de plataformas.
